🌟 Fase 1: Definição de Conjuntos 🌟

Um conjunto é uma coleção de elementos distintos. Ex: A = {a, e, i, o, u}

Qual das opções abaixo representa corretamente um conjunto?

🌈 Fase 2: União 🌈

A união de conjuntos junta todos os elementos, sem repetir.

Se A = {a, b, c} e B = {b, c, d}, qual é A ∪ B?

🍃 Fase 3: Interseção 🍃

A interseção mostra os elementos comuns entre conjuntos.

Se A = {x, y, z} e B = {w, x, z}, qual é A ∩ B?

🧚‍♀️ Fase 4: Complemento 🧚‍♀️

O complemento de A em relação a U (universo) são os elementos de U que não estão em A.

Se U = {1,2,3,4,5} e A = {2,4}, qual é o complemento de A?

🦄 Fase 5: Subconjuntos 🦄

Um conjunto B é subconjunto de A se todos os elementos de B estão em A.

Se A = {1, 2, 3}, qual dos conjuntos abaixo é um subconjunto de A?

🧩 Fase 6: Diferença 🧩

A diferença A - B contém os elementos de A que não estão em B.

Se A = {1,2,3,4} e B = {2,4}, qual é A - B?

⚡ Fase 7: Diferença Simétrica ⚡

A diferença simétrica A △ B são os elementos que estão em A ou B, mas não em ambos.

Se A = {1,2,3} e B = {2,3,4}, qual é A △ B?

💡 Fase 8: Conjunto das Partes 💡

O conjunto das partes (𝒫) de um conjunto é o conjunto de todos os subconjuntos possíveis.

Qual é o conjunto das partes de A = {1, 2}?
